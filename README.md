# Flight Booking Application ✈️

A Salesforce-based Flight Booking Application developed using Lightning Web Components (LWC), Apex, SOQL, and Salesforce Flows.

The application allows customers to:

- Search available flights
- View airline and pricing details
- Book flight tickets
- Receive automated booking confirmation emails
- Manage reservation details through Salesforce automation

---

# Features 🚀

## Flight Search System ✈️
- Search flights based on departure city, arrival city, and departure date
- Dynamic flight listing using Apex and SOQL
- Displays airline details and pricing

## Flight Booking System 🎫
- Book flights using modal popup form
- Stores passenger booking details in Salesforce custom objects
- Displays booking confirmation toast message

## Automated Email Notifications 📧
- Implemented using Salesforce Record-Triggered Flows
- Automatically sends booking confirmation emails to customers after successful booking

## Dynamic UI Using LWC ⚡
- Interactive user interface built with Lightning Web Components
- Conditional rendering for search results and booking popup
- Responsive design using SLDS

## Toast Notifications & Modal Popup 🔔
- Success toast after booking confirmation
- Modal popup for passenger details and booking process
